# sample-api

Simple api for small applications using express, jsonwebtoken, sequelize and websocket

## Installation

The module is now available on npm! `npm i @andrewcaires/sample-api`

## Example usage

```js
import { initialize } from '@andrewcaires/sample-api';

initialize();
```
